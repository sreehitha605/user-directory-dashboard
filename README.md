#  User Directory Dashboard

A simple User Directory Dashboard built using React.  
It allows users to search, sort, and view detailed information of users fetched from a public API.

---

##  Features

-  Fetch users from API
-  Search by name or email
-  Sort users (A-Z, Z-A, Reset)
-  View detailed user information
-  Clean and responsive UI

---

##  Tech Stack

- React (Vite)
- React Router DOM
- CSS

---

##  Installation & Setup

```bash
npm install
npm run dev
